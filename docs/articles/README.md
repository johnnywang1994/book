## 簡介

嗨～歡迎您，我是Johnny，這是我的個人介紹兼紀錄學習點滴的筆記書，內容主要為個人作品展示與學習心得文章。部分內容也會同時發佈在我的 [Daily Learning Instagram 帳號](https://www.instagram.com/johnnycode.daily/)，如果懶得一直來部落格查看但又希望能看到我最新的分享內容，歡迎追蹤關注我喔！

俗話說取之於人，需懂得報恩，從最初開始學習程式設計以來，一直都是靠著網路上的無私分享內容自學，也希望自己能把自己的所學所好分享出去給所有需要的人。

目前在 LINE Taiwan 擔任資深軟體工程師，主要負責前端開發，偶爾會不小心越線碰到些後端，過去特別愛 Vuejs，但小朋友才做選擇，現在我 Vuejs, Reactjs 都愛！平日興趣是窩在家做些酷酷的小專案，透過實作學習讓我獲得很多成就感 =V=，夢想是希望有一天能進步到 Vuejs 作者尤大大 1/10 強。

Be Good, be strong, be yourself, never give up.
