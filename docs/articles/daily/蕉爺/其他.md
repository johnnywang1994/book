# 其他知識

## 人法地，地法天，天法道，道法自然
道就是組成宇宙的龐大能量系統，所以無法用言語描述，因為太玄了，肉眼根本看不見摸不到，而道在進入空間後稱為炁，這個就是張至順道長寫的炁體源流裡講到的那個字，西方稱作以太，而炁在進入地球後被地磁場二分為陰陽兩種磁極，其中天為陽磁場，又稱陽德，剩下的帶電粒子沉重下降到地上，就是陰磁場，而炁本身在地球又受到離他最近的五星影響，金木水火土星，因此炁在進入地球後，陽磁場稱五氣，到達地表陰磁場就被稱為五行，人法地，因此人體本身也跟中醫說的五行直接關聯，這就能解釋整個道家、中醫的一整個體系，道生天，天生地，地生人，一切都是一種能量的推進跟傳導，也因為這樣商朝滅亡前才會追著自己即將殞落的帝星，遷都了13次，道德經裡還有一個道生一，一生二，二升三，三生萬物也跟前面這套吻合，一就是炁，二是陰陽，三是易經說的少陽少陰老陽老陰，萬物就是五行構成的物質世界，而先天八卦之所有跟後天八卦左右倒置，也是因為先天八卦是伏羲他們以地外視角俯瞰地球所說的，而不是從地上視角

## 五德，五行，五氣
德是一種能量的單位，五德對應到仁理信義智，由五德的頻率能量提升，能夠影響五行的運行，而五行的運行又會影響五氣的流動，最終影響人體的健康狀態。這種能量的轉換和流動是道家哲學中的核心概念，強調了宇宙萬物之間的相互關聯和影響。

## 記史不等於歷史
文字無法完全描述出當下的整個狀態，包含情緒、場面、背景等等，也因此文字描述的越詳細，反而與事實越是脫離，根據神形象數理的推論，真正的神完全無法以數理記下，真正的神只有一個循環【成住壞空】
