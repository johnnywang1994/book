# 溫裏劑

- 以溫熱藥物為主組成的方劑，主要用於治療裏虛寒證。
- 裏虛寒證: 指人體內臟功能虛弱，陽氣不足，導致寒邪內生或外寒內侵所引起的證候。
- 溫裏劑適用於裏寒證候，主要表現為：
  - `腹部冷痛`、喜溫喜按
  - `泄瀉清稀`，甚則完穀不化
  - `四肢厥冷`，畏寒蜷臥
  - 脈沉遲或微細
  - `舌淡苔白滑`
- 主要分為幾類，溫中祛寒,回陽救逆,溫經散寒

## 溫中祛寒
- 中焦虛寒證，如`脾胃虛寒`引起的`腹痛、泄瀉`等。

### 理中湯
- 中焦虛寒證。腹痛不渴、`或吐或利`、舌淡苔白或灰黑而滑，脈沉細
- 溫中祛寒，`益氣健脾`。
- 白朮、人參、乾薑、甘草
- 治療中焦虛寒的代表方，脾胃陽虛、寒氣較重，且`伴隨濕盛`（如拉肚子）者
- `濕熱證及陰虛者不適宜。忌生冷水果食物`
- 慢性菌痢、慢性腸炎、慢性胃炎、胃及十二指腸潰瘍、胃下垂
- 比較
  - 病因：理中湯是本身體質偏寒者的常用方劑，通常沒有明顯的誘因（吃冷、空腹、勞累後就發作），若是因飲食不節或外感寒邪等外在因素（吹冷氣、淋雨、天氣悶熱、出入冷熱）而引起的急性腹痛、腹瀉，則不宜使用，應選用`藿香正氣散`或`吳茱萸湯`等方劑。
  - 質地：理中湯,吳茱萸湯通常是清稀不臭，若酸腐、偏臭則應使用`藿香正氣散`。

### 附子理中湯
- 中焦虛寒證。腹痛喜溫喜按，`四肢厥冷`，舌淡苔白，脈沉細
- 溫中祛寒，`補氣健脾`。
- 理中湯+附子
- 脾胃陽虛、寒氣較重，且`四肢厥冷`者
- 此方為補藥，使用時須注意即便有冷汗、惡寒、嘔吐、腹瀉等「假性陽虛」表現，但其真實病機也可能是「濕」或「穢濁之氣」困阻脾胃。若「濕邪」或「食積」未除，用補藥反而會「關門留寇」，讓濕邪更悶在裡面，反而使用`藿香正氣散`等祛濕化濁藥物較為適宜。

### 小建中湯
- `虛勞腹痛綿綿`，時痛時止，溫按則痛減，舌淡苔白，脈細弦而緩
  - 或`心悸`，虛煩不寧，`面色無華`
  - 或四肢痠楚，手足煩熱，咽乾口燥
- 溫中補虛，`和裏`緩急。
- 桂枝、白朮、炙甘草、芍藥、生薑、大棗、飴糖
- 治療`虛寒性腹痛`的代表方，陰陽不和、氣血不足，有疼痛困擾者
- 嘔吐及中滿者慎用。
- `有實熱或血瘀者不宜使用`。
- 比較：
  - 中焦虛寒，但還有一定氣血基礎的人 → 小建中湯
    - 不是痛到受不了，但反覆一直來(虛寒型)
    - 緊張就肚子痛(肝脾不和、腸胃敏感、腸躁)
    - 手腳冰冷、容易肚子痛(小孩、瘦弱者、產後體虛者)
  - 中焦虛寒且氣血不足較嚴重的人 → 理中湯
    - 脾陽大虛、四肢冰冷
    - 腹瀉清稀、精神萎靡
  - 濕重、腹脹明顯要先 化濕 → 藿香正氣散、平胃散
    - 吃了不乾淨的東西、天氣悶熱、吹冷氣後肚子痛腹瀉
    - 噁心想吐、胃脹悶

### 吳茱萸湯
- 胃虛寒證、肝脾虛寒嘔逆、所致的`嘔吐`、`噯氣`、`腹痛`，舌淡苔白，脈沉遲，治陽明，少陰、厥陰三經之病
- 溫中補虛、`降逆止嘔`。
- 吳茱萸、乾薑、人參、甘草
- `胃鬱熱嘔吐、吞酸者禁用`
- 慢性胃炎、消化道潰瘍、頭痛、偏頭痛、嘔吐、妊娠惡阻、急性吐瀉、胃酸過多
- 比較：
  - 寒從肝胃上逆、嘔吐劇烈，肝、胃（上中焦） → 吳茱萸湯
    - 嘔吐劇烈、頻繁
    - 一吐就舒服
    - 痛勢較急
    - `不怕風、不怕冷`
  - 寒濕從外入中、脹悶噁心，脾、胃（中焦） → 藿香正氣散
    - 噁心為主
    - 嘔吐不一定劇烈
    - 吐後不特別舒服
    - 脹悶多於疼痛
    - 頭昏、`微怕冷或怕風`

### 安中散
- 治急慢性胃痛反胃、`泛酸`、寒邪氣滯、停積不消、`腹脅脹痛`及`婦人經痛`。
- 順氣安中、散寒止痛。
- 桂皮,延胡索,牡蠣,小茴香,砂仁,甘草,良薑
- 治脾胃虛寒、氣血凝滯而引起胃痛腹痛之要方。
- 有濕熱甚之炎症性疾病忌用。
- `+ 芍藥甘草湯 = 大正漢方胃腸藥`
- 比較
  - 補氣健脾、溫中祛寒，陽氣不足的寒症 → 理中湯
  - 空腹痛明顯、怕冷、胃痛、腹脹、胃酸過多（口吐酸水） → 安中散
  - 飽脹、吃多不舒服(不一定痛)、大便臭 → 保和丸



## 溫中祛寒辨證
- 手腳冰冷，完榖不化=> 理中湯
- 四肢厥冷腹痛=> 附子理中湯
- 心悸、虛勞腹痛綿綿不定=> 小建中湯
- 嘔吐劇烈、噯氣腹痛=> 吳茱萸湯
- 胃痛、腹脹、胃酸過多=> 安中散


## 回陽救逆
- 用於治療陽氣衰微、陰寒內盛的危重證候，如四肢厥逆、脈微欲絕等。

### 四逆湯
- 陽氣衰微、陰寒內盛所致的`四肢厥逆`，惡寒踡臥，神疲欲寐，`脈微欲絕`，面色蒼白，舌淡苔白，`汗出不止`
- 溫經逐寒，`回陽救逆`。
- 附子、乾薑、甘草
- 治療陽氣虛衰、陰寒內盛的代表方，四肢厥冷、脈微欲絕者，誤汗或大汗淋漓所致亡陽危證，尤宜急用本方
- 人已處休克邊緣，純粹為了【點火】救命用，`熱病後虛證者慎用`
- 急性心肌梗塞、休克、中風昏迷、急性腎上腺皮質功能不全

> 跟四逆散是完全不同東西


## 溫經散寒
- 用於治療寒邪凝滯經脈所致的疾患，如經寒不調、宮寒不孕等。

### 當歸四逆湯
- `血虛陽氣不足`而感受`寒邪`諸證。`手足厥冷`、`腰腿痺痛`或`局部青紫`、舌淡苔白、`脈沉細`。
- `養血散寒`，溫經通脈。
- 當歸、桂枝、白朮、芍藥、乾薑、通草、炙甘草
  - 成分和四逆湯差異很大
- 治療`寒邪凝滯經脈`所致的`婦科疾患的代表方`，經寒不調、宮寒不孕者
- 陽氣鬱所致之熱厥、陰虛火旺者慎用
- `增強血液循環`，消炎。用於血栓閉塞性脈管炎、神經血管性頭痛、肩周炎、關節痛、坐骨神經痛、胃及十二指腸潰瘍、`慢性蕁麻疹`、心絞痛、心肌梗塞、凍傷、手掌角皮症

### 陽和湯
- `陰疽初起`，多陰平素陽虛，陰寒之邪乘虛侵襲，阻于筋骨血脈之中，致`血虛寒凝痰滯`而成，患處漫腫、皮色不變，`酸痛`無熱、舌淡苔白、脈沈細或沈遲
- 溫經散寒，`活血化瘀`。
- 當歸、川芎、赤芍、細辛、乾薑、桂枝、黃芩、炙甘草
- 治療`陰疽`的代表方
- 瘡瘍`紅腫熱痛屬陽證者、陰疽已潰者不宜使用`
- 初期乳腺炎、疔瘡初起

> 陰疽的「溫散」與疔瘡的「清熱」在初期截然不同，但中後期都會走向「托膿、生肌」的共性治療，只是用藥不同
