# 和解劑

- 調和陰陽、平衡臟腑
- 專門處理邪氣在半表半裡（如`少陽`）或臟腑功能失和（如`肝脾不和、寒熱錯雜`）的病證
- 和解少陽,調和肝脾,調和腸胃

## 常見於以下人群
- 壓力大、情緒波動大: 肝氣鬱結 → 少陽氣機失調
- 作息不規律、生活節奏大起大落: 氣機調節功能下降，樞機不利
- 飲食不節: 肝膽鬱熱、影響脾胃 → 少陽不利
- 時常半生半病狀態: 典型的 少陽“半表半裡” 特質
  - 常覺得有點不舒服但又不像真正「感冒」
- 容易被外邪侵襲但沒有完全發作: 外邪入裏未入、未出，卡在少陽

### 和解少陽 vs 調和肝脾
➡ 少陽問題比較像 `外邪或體內壓力卡在樞機，進退不得`。
➡ 肝脾問題比較像 `情緒與壓力讓能量不暢，影響消化功能`。


## 和解少陽
- 邪在足少陽膽經
- `往來寒熱`，胸脅苦滿，心煩喜嘔，`不欲飲食`，以及`口苦、咽乾、目眩`
- 小柴胡湯、大柴胡湯、柴胡桂枝湯、柴胡加龍骨牡蠣湯
- 常是這樣開始的
  - 感冒後拖著拖著沒完全好
  - 病情反覆，覺得整個人「卡住」，情緒上比較煩躁、急、容易緊張

### 小柴胡湯
- 傷寒少陽證，往來寒熱，胸脅苦滿，`納呆，心煩喜嘔`，`口苦咽乾耳聾`，`脈弦數`，舌苔淡白
- `和解少陽`，疏肝和胃
- 柴胡、黃芩、半夏、人參、甘草、生薑、大棗
- 調整肝臟和膽囊、膽管、奧狄括約肌的活動，調整中樞神經系統的活動，調理胃腸的蠕動，增強免疫功能。用於急慢性肝炎、急慢性胃炎

### 大柴胡湯
- 少陽兼陽明實證，`胸脅苦滿，心煩喜嘔`，`大便痞滿`，`脈弦實`，舌苔黃厚，脈弦有力
- 外解表邪，內通裡實。
- 小柴胡湯-人參、甘草，+大黃、枳實、芍藥
- 用於肝膽疾病合併便秘者，如肝膽炎症、膽石症、慢性便秘等，`非實證者慎用`

### 柴胡桂枝湯
- 外邪入少陽兼太陽證，`發熱惡寒，無汗頭痛，四肢煩疼`，胸脅苦滿，心煩喜嘔，`脈弦緊`
- `表裡兩解`、寒熱兼除
- 小柴胡湯+桂枝湯
- 用於外感`風寒`，`邪入少陽`證，兼有太陽表證者
- 感冒、流感、胰腺炎、急慢性胃炎、慢性肝炎、膽結石

### 柴胡加龍骨牡蠣湯
- 少陽失調、痰擾心神，胸脅苦滿，心煩喜嘔，`失眠多夢，健忘驚悸`，`脈弦細`
- 和解少陽、`重鎮安神`
- 小柴胡湯+龍骨、牡蠣、茯苓、大黃、桂枝
- 用於少陽失調、痰擾心神所致的`情志不寧`、`失眠多夢`、`健忘驚悸`等症
- 動脈硬化症、腦溢血、心臟瓣膜疾病、梅尼爾氏症、精神分裂症

### 和解少陽辨識
往來寒熱，胸脅苦滿，心煩喜嘔
- 口苦咽乾，小柴胡湯
- 大便秘結，大柴胡湯
- 發熱惡寒無汗，柴胡桂枝湯
- 失眠多夢健忘，柴胡加龍骨牡蠣


## 調和肝脾
- `肝氣鬱結，橫犯脾胃`，或脾虛不運，影響肝不疏泄
- `胸悶脅痛，脘腹脹痛`，不思飲食，大便泄瀉，甚則寒熱往來
- 常是這樣開始的
  - 工作壓力、情緒鬱悶造成
  - 生悶氣 → 肚子脹
  - 吃不好、睡不好
  - 女生月經容易受影響

### 四逆散
- 肝鬱脾胃不和，胸脅苦滿，手足厥冷，但身熱或脘腹疼痛，食慾不振，`腹瀉或便秘`，或小便不利，`脈弦`
- `透解鬱熱`，疏肝理脾
- 柴胡、白芍、枳實、甘草
- 用於肝氣鬱結，脾胃功能失和所致的胸脅苦滿、脘腹脹痛、食慾不振、腹瀉或便秘等症
- 慢性胃炎、腸易激綜合症、慢性肝炎、膽囊炎、膽石症
- `脾腎陽衰、陰邪厥逆者禁用`

### 逍遙散
- 肝鬱血虛、脾氣不足，胸脅苦滿，`情緒抑鬱`，頭暈目眩，`食慾不振`，腹脹，`月經不調`，或經量過多，`脈弦細`
- 疏肝解鬱，`健脾養血`
- 柴胡、白芍、當歸、白朮、茯苓、甘草、生薑、大棗
- 用於肝氣鬱結，`血虛`，脾氣不足所致的胸脅苦滿、情緒抑鬱、食慾不振、月經不調等症
- 慢性肝炎、慢性胃炎、月經不調、更年期綜合症

### 加味逍遙散
- `肝鬱化火`，胸脅苦滿，`情緒煩躁易怒`，頭暈目眩，`口苦咽乾`，食慾不振，腹脹，`月經不調`，或經量過多，`脈弦數`
- 疏肝`清熱`，健脾養血
- 逍遙散+牡丹皮、梔子
- 用於`肝氣鬱結化火`所致的胸脅苦滿、情緒煩躁易怒、口苦咽乾等症
- 慢性肝炎、慢性胃炎、肝膽疾病

### 調和肝脾辨識
胸脅苦滿，脘腹脹痛，不思飲食
- 腹瀉或便秘，四逆散
- 情緒抑鬱，逍遙散
- 情緒煩躁易怒，加味逍遙散


## 調和腸胃
- 邪犯腸胃，寒熱夾雜，升降失常，而致心下痞滿，噁心嘔吐，`脘腹脹痛`，腸鳴下利
- 半夏瀉心湯、甘草瀉心湯、生薑瀉心湯
- 常是這樣開始的
  - 飲食不節，暴飲暴食
  - 吃了不合胃口的東西
  - 消化功能失調，容易噁心嘔吐、腹瀉

### 半夏瀉心湯
- 寒熱錯雜，心下`痞滿`，反酸、噁心嘔吐、打嗝，脘腹脹痛，`便秘與腹瀉交替`，或`腹鳴下利`，`脈弦`
- 和胃降逆，`散結消痞`
- 半夏、黃芩、黃連、乾薑、人參、甘草、大棗
- 用於寒熱錯雜，胃腸「卡住」，吃下去像堵著，`腸胃功能失調所致的心下痞滿`、噁心嘔吐、脘腹脹痛、大便溏薄等症
- 慢性胃炎、腸易激綜合症、消化不良
- 常見情境
  - `「寒熱錯雜」的腸胃問題`時(最常見)
    - 上有熱、下有寒
    - 胃部脹、硬、痛（熱、鬱滯）
    - 腸胃又虛弱、容易瀉（寒、虛）
    - 上下不協調 → 就叫「寒熱錯雜」
  - `用藥錯誤後的副作用`
    - 過用攻下或清熱的後遺症治療
    - 誤治／過治 → 損傷胃氣 → 上熱下寒 → 寒熱錯雜
  - 胃腸「功能性障礙」但不屬於單純虛或實
    - 現代人因壓力、情緒、飲食失調而出現
    - 吃太快、太油
    - 情緒波動，長期胃弱 → 偶爾又燥熱內生
    - 反覆胃脹但不是純寒也不是純熱的症狀
  - `明明脹得像有火，但吃一點涼就拉，反酸、噁心、痞滿`
    - 這種「寒熱錯雜」的胃腸症狀，半夏瀉心湯最適合
