# 治燥劑
- 主治燥邪犯肺所致的咳嗽、氣喘、胸痛、口乾、咽痛、皮膚乾燥等症狀。
- 常用的治燥劑分清宣潤燥, 滋養潤燥。


## 清宣潤燥
- 主要用於外感燥邪犯肺的症狀, 如咳嗽、氣喘、胸痛、口乾、咽痛等。
- 常用的藥物有桑葉、菊花、杏仁、枇杷葉等。

### 杏蘇散
- `外感風寒`證。症見`發熱惡寒`、頭痛無汗、咳嗽痰稀而白、`鼻塞咽乾`、`苔白薄`、脈浮或弦
- 發散風寒、宣肺化痰
- 紫蘇,桔梗,半夏,枳殼,橘皮,茯苓,杏仁,前胡,甘草,大棗,生薑
- 感冒、流感、支氣管炎、咳嗽、慢性鼻炎。

### 桑杏湯
- `秋感溫燥`，灼傷肺津，身不甚熱，`乾咳無痰`，咽乾口渴，舌紅，苔薄白而燥，`右脈數大`者
- 清宣燥熱，潤肺止咳
- 桑葉,杏仁,沙蔘,象貝,香豉,梔皮,梨皮

### 清燥救肺湯
- `溫燥傷肺`證。頭痛`身熱`，乾咳無痰、`氣逆而喘`、口乾而渴、`鼻燥咽乾`、舌`無苔而乾`，脈虛大而數
- 清燥潤肺。
- 桑葉,石膏,人參,胡麻仁,阿膠,麥冬,杏仁,枇杷葉,甘草
- 脾胃虛寒者慎用
- 肺炎、肺結核、支氣管哮喘、急慢性支氣管炎、肺氣腫、肺癌、蕁麻疹、喉頭炎

### 沙參麥冬湯
- `燥傷肺胃，津液虧損`證。咽乾口渴，`乾咳少痰`，舌紅少苔，`脈細數`
- 清養肺胃、生津潤燥
- 沙參,麥冬,玉竹,生甘草,冬桑葉,生扁豆,花粉
- 慢性支氣管炎、慢性肺炎、慢性咽炎、慢性胃炎、小兒口瘡、消化道潰瘍

### 清宣潤燥比較
- 發熱惡寒: 杏蘇散
- 乾咳無痰,身不甚熱: 桑杏湯
- 氣逆而喘,頭痛身熱: 清燥救肺湯
- 乾咳少痰,咽乾口渴: 沙參麥冬湯


## 滋養潤燥
- 主要用於內傷燥邪的症狀, 如皮膚乾燥、口乾、咽痛等。
- 常用的藥物有沙參、麥冬、玉竹、百合等。

### 增液湯
- 陽明溫病，無上焦證，數日不大便，其陰素虛，不可用承氣湯者
- 增液潤腸

### 麥門冬湯
- `肺胃陰虛`證，乾咳少痰或痰黏不易咳出，咽乾口燥，舌紅少苔，脈細數
- 潤肺養陰，清胃生津。
- 麥門冬,人參,五味子,粳米,甘草,桔梗,貝母,生薑
- 外感風寒咳嗽，`痰多清稀者不宜`使用
- 慢性咽炎、支氣管炎、失聲、肺結核、肺癌

### 百合固金湯
- `肺腎陰虛、虛火上炎`證，咳嗽氣喘，`痰中帶血`，`乾咳無痰`，咽乾口燥疼痛，`潮熱盜汗`，五心煩熱，舌紅少苔，`脈細數`
- `養陰清熱`、潤肺化痰
- 生地黃,熟地黃,麥冬,百合,芍藥,當歸,貝母,元蔘,桔梗
- 用於肺腎陰虛兼有虛熱證者，咳嗽痰血的常用方
- 咳痰帶血`肺有實熱者，不宜使用`，脾虛便溏者慎用
- 肺結核、肺癌、支氣管擴張咯血、肺炎後期、慢性肝炎、慢性咽炎、慢性咽痛
> 咳嗽氣逆、失聲、支氣管炎 -> 麥門冬湯

### 滋養潤燥比較
- 便祕: 增液湯
- 乾咳少痰或痰黏不易咳出、失聲: 麥門冬湯
- 咳嗽氣喘，痰中帶血，乾咳無痰，咽乾口燥疼痛，潮熱盜汗，五心煩熱: 百合固金湯
