# 補益劑

- 以補益藥物為主組成的方劑，主要用於治療各種虛證。
- 虛證是指人體正氣不足，臟腑功能衰弱所引起的證候。
- 補益劑適用於虛證候，主要表現為：
  - 氣虛：乏力、懶言、氣短、自汗、脈虛弱
  - 血虛：面色蒼白、眩暈、心悸、失眠、舌淡脈細
  - 陰虛：潮熱、盜汗、口乾咽燥、舌紅少苔、脈細數
  - 陽虛：畏寒肢冷、腰膝酸軟、小便清長、舌淡苔白、脈沉遲
- 補益劑主要分為以下幾類：
  - 補氣：用於治療氣虛證，如四君子湯等。
  - 補血：用於治療血虛證，如四物湯等。
  - 補陰：用於治療陰虛證，如六味地黃丸等。
  - 補陽：用於治療陽虛證，如金匱腎氣丸等。


## 補氣

### 四君子湯
- `脾胃氣虛`，心腹脹滿，`面色蒼白`，語弱聲低，四肢無力，納呆，`脈虛無力`，舌淡苔白
- `甘溫益氣`，健脾養胃
- 人參,白朮,茯苓,甘草
- 健脾的基礎方劑
- 增強消化吸收功能，提高機體免疫力
- 慢性胃腸炎、胃下垂、胃弛緩、胃十二指腸潰瘍、胃腸功能減退、手足痿弱、半身不遂、糖尿病、夜尿、遺尿

### 六君子湯(四君子湯+二陳湯)
- 脾胃`氣虛兼痰濕`之證。`食少便溏`，胸脘痞悶，甚則嘈雜，咳嗽`痰多色白`，`噁心嘔吐`，`舌淡苔白，脈濡緩`。
- 益氣健脾，`燥濕化痰`。
- 人參、白朮、茯苓、甘草、陳皮、半夏

### 香砂六君子湯
- 脾胃虛弱，`濕阻氣滯`，`虛寒腹痛`，`胸脘痞悶`，食後`腹脹`，不思飲食，嘔吐噯氣，`泛酸`，`苔白膩，脈濡弱`。
- 益胃補中，`理氣和胃`。
- 人參、白朮、茯苓、甘草、陳皮、半夏、木香、砂仁
- `抗潰瘍`，增強胃腸功能，增強免疫功能，止痛。
- 常用於胃炎、胃、十二指腸潰瘍、胃弛緩、胃下垂、胃癌、胃酸過多、慢性腹膜炎、食慾不振、氮血症
  - `腹瀉、倦怠`、面色萎黃等脾虛濕盛表現為主，則可選`參苓白朮散`(無氣滯)
  - 同時有`腹脹、噯氣、消化不良`等氣滯表現，則可選此方(氣滯)

### 參苓白朮散
- `脾胃氣虛夾濕`證，`四肢乏力`，`食少便溏`，形體消瘦、飲食不化或吐或瀉胸脘悶脹，面色萎黃、舌淡苔白膩，`脈虛緩`
- 益氣健脾，`和胃滲濕`
- 人參,白朮,茯苓,甘草,扁豆,山藥,蓮子,薏苡仁,桔梗,砂仁,大棗
- 增強消化吸收能力，增強機體免疫功能，`止瀉`
- 用於慢性胃腸炎，`病後腸胃調理`
- 偏溫補，若睡前空腹吃肚子抽痛，建議白天、飯後吃

### 補中益氣湯
- 脾胃虛弱導致的體倦乏力、食少腹脹、`久瀉`、`脫肛`、`子宮脫垂`
- 補中益氣、`升陽舉陷`
- 人參,白朮,炙甘草,黃耆,當歸,陳皮,升麻,柴胡
- 治療`氣虛發熱`、自汗、內臟下垂
- 偏溫燥，`口乾舌燥、陰液津虧、陰虛發熱或內熱熾盛者忌用`
- 若使用錯誤導致上火吞嚥痛，可考慮用沙參麥冬湯調理

### 生脈散
- 暑熱多汗、耗氣傷陰證。
  - 體倦、氣短懶言、咽乾口渴，脈虛弱。
  - 久咳肺虛、氣陰兩傷證。嗆咳少氣、短氣自汗、口舌乾燥，脈細。
- `益氣生津`，斂陰止汗
- 人參、麥冬、五味子
- 生津止渴、益氣固表
- 用於暑熱傷津、氣陰兩傷，或久病體虛
- 以補益為主，輔助身體恢復正氣，並非直接清除暑熱，`實熱尚在、邪熱未清者不可單用`

### 補氣劑選擇指南
- 純氣虛(無濕無滯): 四君子湯
- 氣虛兼痰濕: 六君子湯
- 氣虛兼濕阻氣滯: 香砂六君子湯
- 氣虛夾濕偏滲濕止瀉: 參苓白朮散
- 氣虛下陷: 補中益氣湯
- 氣陰兩傷: 生脈散


## 補血
### 四物湯
- `血虛`證、衝任虛損證，貧血、`面色萎黃`，`頭暈`目眩，`心悸`失眠，`月經量少或閉經`，舌淡脈細
- `補血調經`，養肝活血
- 當歸,川芎,白芍,熟地黃
- 補血的基礎方劑
- 脾胃虛弱，運化無力，食少便溏者，慎用

### 八珍湯(四君子湯+四物湯)
- `氣血兩虛`證。面色萎黃，頭暈目眩，心悸失眠，`四肢乏力`，`食少便溏`，舌淡苔白，脈虛弱
- `補氣養血`，健脾和中
- 人參,白朮,茯苓,甘草,當歸,川芎,白芍,熟地黃
- 補氣補血的基礎方劑
- 用於貧血、月經不調、產後虛弱、`體虛乏力`、久病失治等

### 歸脾湯
- `心脾氣血兩虛`引起的證狀，包括`心悸`、`失眠`、`食少體倦`、`健忘`、`盜汗`虛熱，以及脾虛導致的便血、月經不調(量多色淡、或淋漓不止、或帶下)，舌淡、苔白薄，`脈細緩`
- `健脾寧心`、益氣補血
- 人參,白朮,茯苓,炙甘草,黃耆,當歸,桂圓肉,酸棗仁,遠志,木香,生薑,大棗
- 改善消化不良、食慾不振
- 滋補心神，改善心悸、焦慮、失眠
- 增強氣血，改善氣血不足導致的面色萎黃、`體倦無力`
- 對於脾虛不能攝血導致的月經不調、`崩漏`等也有治療作用
- 主治健忘、`體倦食少，伴有消化功能弱`
- `內熱的出血證`或`失眠證`，`慎用或禁用`本方

### 養心湯
- `心血虛少`、神經衰弱、神志不寧、`驚悸怔忡`、或神倦、失眠、面色蒼白，主治心悸、失眠嚴重，伴有`心神不安`(消化系統還算正常)。
- 養心補血、`安神定志`。
- 人參,黃耆,當歸,熟地黃,茯苓,遠志,酸棗仁,柏子仁,炙甘草
- 比較(都處理失眠，但針對的虛不同)
  - 血少、血不養神，補血為主，易醒、頭暈、舌淡脈細 -> 養心湯
  - 心腎陰虧、虛火擾神，滋陰為主、虛熱型失眠，多夢、心煩、口乾、盜汗、舌紅少苔脈細數 -> 天王補心丹
  - 脾氣虛、心失血養，過勞思慮型失眠，食慾差、疲倦、便溏、舌淡胖脈虛弱 -> 歸脾湯
  - 氣虛血少、心動悸、虛煩失眠、自汗盜汗，舌光少苔脈虛數 -> 炙甘草湯

### 炙甘草湯
- 氣虛血少，`心動悸`，胸悶氣短，舌光少苔，`脈結代`。
- 虛勞`肺痿，乾咳無痰`，或`喀痰不爽`，痰中帶血，`虛煩失眠`，`自汗盜汗`，咽乾口燥，便乾，`脈虛數`。
- 益氣滋陰，養血復脈。
- 炙甘草,人參,桂枝,生地黃,阿膠,麥冬,麻仁,生薑,大棗


### 補血劑選擇指南
- 體力透支(身體累): 八珍湯
- 心力交瘁(腦累、睡不好、胃口差): 歸脾湯
- 心悸失眠嚴重(腦累、胃口無特別差): 養心湯
- 心動悸、心律不整、虛煩失眠、自汗盜汗: 炙甘草湯

### 補血劑失眠型態比較
| 項目    | 養心湯 | 天王補心丹   | 歸脾湯   | 炙甘草湯   |
| ----- | --- | ------- | ----- | ------- |
| 虛的核心  | 血虛  | 陰虛      | 氣血兩虛  | 氣虛血少  |
| 有沒有虛熱 | ❌   | ⭕       | ❌     | ❌      |
| 口乾    | 少   | 明顯      | 少     | 少      |
| 舌象    | 淡   | 紅少苔     | 淡胖    | 光少苔   |
| 失眠型態  | 易醒  | 多夢、煩    | 累但睡不好 | 虛煩失眠  |
| 適合長期  | ⭕   | ⚠️（看脾胃） | ⭕     | ⭕      |



## 補陰
### 六味地黃丸
- `肝腎陰虛`證，`腰膝酸軟`，`頭暈耳鳴`，`盜汗`，骨蒸潮熱，五心煩熱，口乾咽燥，舌紅少苔，`脈細數`
- 滋補肝腎。
- 熟地黃,山藥,山茱萸,澤瀉,牡丹皮,茯苓
- 補腎滋陰的基礎方劑
- 腎陰虛證常見於`更年期綜合症`、`慢性腎炎`、`糖尿病`等疾病的治療中
- 脾胃虛弱者服之易致胃腸脹氣、消化不良、軟便，須搭配健脾胃方藥一起服用
- 慢性腎炎，高血壓，糖尿病，神經衰弱，婦女更年期綜合症，抗心律失常，慢性前列腺炎，遺尿症，中心性視網膜炎及視神經炎，紅斑性狼瘡，肺結核，甲狀腺機能亢進

### 知柏地黃丸(六味地黃丸+知母+黃柏)
- `肝腎陰虛`，`虛熱內盛`證，腰膝酸軟，頭暈耳鳴，`五心煩熱`，`潮熱盜汗`，口乾咽燥，舌紅少苔，`脈細數`
- 滋補肝腎，`清虛熱`
- 熟地黃,山藥,山茱萸,澤瀉,牡丹皮,茯苓,知母,黃柏
- 用於肝腎陰虛兼有虛熱證者

### 杞菊地黃丸(六味地黃丸+枸杞子+菊花)
- `肝腎陰虛`，`肝陽上亢`證，腰膝酸軟，頭暈耳鳴，`目赤腫痛`，視物昏花，口乾咽燥，舌紅少苔，`脈細數`
- 滋補肝腎，`平抑肝陽`
- 熟地黃,山藥,山茱萸,澤瀉,牡丹皮,茯苓,枸杞子,菊花
- 用於肝腎陰虛兼有肝陽上亢證者

### 百合固金湯
- `肺腎陰虛、虛火上炎`證，咳嗽氣喘，`痰中帶血`，`乾咳無痰`，咽乾口燥疼痛，`潮熱盜汗`，五心煩熱，舌紅少苔，`脈細數`
- `養陰清熱`、潤肺化痰
- 生地黃,熟地黃,麥冬,百合,芍藥,當歸,貝母,元蔘,桔梗
- 用於肺腎陰虛兼有虛熱證者，咳嗽痰血的常用方
- 咳痰帶血`肺有實熱者，不宜使用`，脾虛便溏者慎用
- 肺結核、肺癌、支氣管擴張咯血、肺炎後期、慢性肝炎、慢性咽炎、慢性咽痛
> 咳嗽氣逆、失聲、支氣管炎 -> 麥門冬湯

### 益胃湯
- `胃陰虛`證，`乾咳身熱`、`胃口極差`，舌紅光滑，慢性萎縮性胃炎
- 滋養胃陰。
- 生地黃,麥冬,生地,玉竹,冰糖
- 純粹的滋補劑，專門修復胃腑因`熱病後流失的津液`，不具備散熱功能
- 沙參麥冬湯(治燥劑)不僅滋陰，更具備較強的「清散燥熱」能力，適用於燥氣傷及肺胃
- 舌苔厚膩（有痰濕）或正在拉肚子（脾虛便溏）忌用

### 天王補心丹
- `心腎陰虛`，`虛火擾神`證，健忘怔忡，心煩少寐，心悸神疲，夢遺失精，`口舌生瘡`，`大便秘結`，舌紅少苔，`脈細數`
- 養心安神、滋陰`清虛熱`。
- 生地黃,麥冬,天冬,人參,玄參,丹參,桔梗,當歸,五味子,酸棗仁,柏子仁,遠志,茯苓
- 用於心腎陰虛兼有虛熱證者，失眠多夢的常用方
- 較滋膩，`脾虛便溏者慎用`
- 神經衰弱、精神官能症、精神分裂症、更年期綜合症、口腔炎、心臟瓣膜疾病、心律失常、蕁麻疹、健忘、夢遺、心神不寧


### 補陰劑選擇指南
- 肝腎陰虛無熱: 六味地黃丸
- 肝腎陰虛兼虛熱: 知柏地黃丸
- 肝腎陰虛兼肝陽上亢: 杞菊地黃丸
- `肺腎陰虛`兼虛火上炎: 百合固金湯
- 胃陰虛: 益胃湯



## 補陽
### 八味地黃丸(八味腎氣丸,金匱腎氣丸)
- `腎陽虛`證，`腰膝痠軟、畏寒肢冷`、`自汗`、`夜尿`、水腫、消渴、陽痿早泄、`耳鳴`，整個人感覺「冷」且虛弱，夜尿多
- 溫養下焦，補益腎陽。
- 熟地黃,山藥,山茱萸,澤瀉,牡丹皮,茯苓,肉桂,附子
- 腎陽虛證常見於`慢性腎炎`、`遺尿症`等疾病的治療中
- 脾胃虛弱者服之易致胃腸脹氣、消化不良、軟便，須搭配健脾胃方藥一起服用
- 口渴尿多，口乾舌燥、喉嚨痛、發燒等「陰虛火旺」或實熱證者禁用。
- 慢性腎炎、腎硬變、腎結石、腎結核、蛋白尿、閉尿、尿失禁、夜尿症、膀胱結石、膀胱結核、前列腺肥大、下肢麻痺、神經衰弱、健忘症、夢遺、早洩、陰萎

### 濟生腎氣丸
- `腎陽虛`證，`明顯水腫`，腰重腳腫，`小便不利`，畏寒肢冷，腰膝痠軟，陽痿、`夜尿頻數`，舌淡苔白，脈沉
- 補腎助陽，`利水消腫`。
- 熟地黃,山藥,山茱萸,澤瀉,牡丹皮,茯苓,肉桂,附子,牛膝,車前子
- 陰虛火旺、`實熱傷津者慎用`
- 慢性腎炎、遺尿症、陽痿、慢性前列腺炎、慢性支氣管炎、血壓異常、腳氣、健忘症、坐骨神經痛

### 補陽劑選擇指南
- 腎陽虛火衰: 八味地黃丸
- 腎陽虛下肢水腫: 濟生腎氣丸
