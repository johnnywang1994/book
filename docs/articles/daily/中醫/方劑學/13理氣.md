# 理氣劑
- 氣郁而肝失條達，則見`胸膈痞悶`
- 氣鬱又使血行不暢而成血鬱，故見`胸脅脹痛`
- 氣血鬱久化火，則見`噯腐吞酸`吐苦之火鬱
- 氣郁即肝氣不舒，肝病及脾，脾胃氣滯，運化失司，升降失常，則`聚濕生痰`，
- 食滯不化而見`噁心嘔吐`。
- 血鬱偏重者，重用川芎，酌加桃仁、赤芍、紅花等以助活血祛瘀；
- 濕鬱偏重者，重用蒼術，酌加茯苓、澤瀉以助利濕；
- 食鬱偏重者，重用神曲，酌加山楂、麥芽以助消食；
- 火鬱偏重者，重用山梔，酌加黃芩、黃連以助清熱瀉火；
- 痰鬱偏重者，酌加半夏、瓜蔞以助祛痰。


## 行氣
- 行氣劑以行氣解鬱為主，兼有疏肝、健脾、化痰、清熱等作用，適用於氣滯血瘀、痰濁內阻、食積不化等證候。

### 越鞠丸(六鬱全才)
- 治氣、血、痰、火、濕、食`六鬱`之證。胸膈痞悶，脘腹脹痛，`吞酸嘔吐、飲食不化`。
- 行氣解鬱
- 蒼术、香附、川芎、梔子、神曲
- 虛證鬱滯，不宜單獨使用
- 調整胃腸功能，利膽，抑菌，消炎。用於慢性胃炎、胃及十二指腸潰瘍、慢性肝炎、膽囊炎、膽石症、消化不良

### 半夏厚朴湯
- 七情鬱結，梅核氣，`咽中如有物阻`，吐之不出、吞之不下，胸脅滿悶，或咳或嘔，舌苔`白膩`，脈弦滑
- 行氣散鬱，`降逆化痰`
- 半夏、厚朴、茯苓、蘇葉、生薑
- 陰虛有熱者慎用

### 厚朴溫中湯
- 脾胃`寒濕、氣機阻滯之脘腹脹滿`，或客寒犯胃、時作疼痛，`不思飲食`，四肢`倦怠`、`泛吐清水`，舌淡`苔白膩`，脈沈。
- `溫中理氣`、燥濕除滿。
- 厚朴,陳皮,甘草,草荳蔻,茯苓,木香,乾薑,生薑
- 健胃，改善消化系統功能，鎮痛。用於急慢性胃炎、胃、十二指腸潰瘍、急慢性肝炎、慢性腸炎、`白帶`、消化不良

### 行氣劑比較
- 廣泛的「悶、脹、酸」，不清楚到底是吃太多、還是壓力大、或濕氣重 => 越鞠丸
- 情緒壓力造成的痰氣互結(梅核氣) => 半夏厚朴湯
- 肚子又脹又冷，整個人沒力氣，還一直吐清水，寒多0於鬱 => 厚朴溫中湯


## 降氣
- 降氣劑以降氣平喘為主，兼有祛痰、清熱、理氣等作用，適用於上盛下虛、痰熱蘊肺、胃虛有熱等證候。

### 蘇子降氣湯
- `喘咳短氣`，痰涎壅盛，`胸膈滿悶`，上盛下虛；或`腰疼腳弱`肢體`怠倦`，或`肢體浮腫`，舌苔`白滑或白膩`。
- 半夏,蘇子,前胡,厚朴,肉桂,當歸,生薑,甘草,大棗
- 降氣平喘，`袪痰止咳`。
- 降氣與納氣同用，以降氣為主，使氣降痰消，則喘咳自平
- 忌生冷、肥膩飲食、避風寒
- 肺腎兩虛之喘咳及肺熱痰喘均不宜

### 定喘湯
- `哮喘證`。喘咳氣急，`痰稠色黃`，胸膈脹悶或`發熱惡寒`，`苔黃膩`，脈滑數。
- 宣肺清熱，`化痰定喘`
- 麻黃,桑白皮,白果,黃芩,蘇子,杏仁,半夏,款冬花,甘草
- 本方排痰，`解除支氣管痙攣`，抑菌，消炎。用於氣喘、咳嗽、支氣管炎、肺氣腫、感冒
- 麻杏甘石湯是純肺熱，大量石膏清肺熱
  - 本方適用於痰熱蘊肺(寒熱錯雜)，清熱化痰藥物較多，適用於痰熱較重者

### 橘皮竹茹湯
- 久病體弱或吐下後`胃虛有熱`，氣逆不降，`呃逆或嘔吐`，舌嫩紅，`脈虛數`。
- 理氣降逆，`益胃清熱`。
- 橘皮,竹茹,大棗,黨篸,生薑,甘草
- 促進胃蠕動，止嘔吐。慢性胃炎、`妊娠反胃`、幽門不完全性梗阻，見呃逆，嘔吐

### 降氣劑比較
- `肺上盛腎下虛`(頭重腳輕)，喘咳、痰多、腰痠腳軟，想降氣又想暖腳 => 蘇子降氣湯
- `肺痰熱`，哮喘證(感冒後突然誘發的急性氣喘)，氣喘到不行、痰黃稠 => 定喘湯
- `胃虛熱`，口渴、呃逆（打嗝）、嘔吐，`打嗝不停、胃虛虛` => 橘皮竹茹湯
