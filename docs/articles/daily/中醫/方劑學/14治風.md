# 治風劑
- 「風」被視為百病之長，其特性是善行而數變（發病快、部位不固定）。
- 專門用來疏散、平息「風邪」的方劑
- 主要分為兩大類：疏散外風與平熄內風

## 疏散外風
- 治療外感風邪引起的疾病，如感冒、頭痛、鼻塞、咳嗽等，或者風邪侵襲經絡引起的疼痛、麻痺、濕熱等症狀。
- 常見於風邪侵襲表層、經絡或肌膚，或者風邪與其他邪氣（如寒、熱、濕）相搏的證型。

### 川芎茶調散
- 外感`風邪頭痛`。症見`頭痛久而不止`，其痛或偏或正，或巔頂作痛，或見惡寒發熱，鼻塞痰盛，頭暈目眩、衝風淚出，時流清涕，舌苔薄白、脈浮
- 疏散風邪，`清熱止痛`。
- 薄荷,防風,荊芥,川芎,細辛,羌活,白芷,甘草
- 鎮痛，鎮靜，抗炎，解熱，增強機體(腦)耐缺氧能力，感冒、正偏頭痛、過敏性鼻炎、鼻竇炎、頭暈、鼻塞、痰盛、嗜睡症

### 大秦艽湯
- 風邪初中經絡，`手足不能`自主活動，`舌強`不能言，風邪散見，不拘一經
- 祛風清熱，調理氣血，`養血榮筋`
- 秦艽,甘草,川芎,當歸,白芍,羌活,防風,細辛,黃芩,石膏,白芷,白术,生地,熟地,茯苓,獨活
- 中風後遺症，半身麻痺，貧血體質，言語障礙，消炎、鎮痛、解熱、抗凝血、改善血液循環、降壓。用於神經麻痺、上下肢神經痛、中風後遺症、半身麻痺、言語蹇澀、顏面神經麻痺

### 消風散
- 風濕侵淫血脈，致生`瘡疥，瘙痒`不絕，皮膚瘙痒或水液流溢，及大人小兒`風熱癮疹`，偏身雲班點，乍有乍無者，風和濕熱相搏，內不得疏泄，外不得透達，鬱于肌膚腠理之間而發
- `養血祛風`，清熱燥濕。
- 荊芥,防風,牛蒡子,知母,蟬蛻,蒼朮,苦參,石膏,當歸,生地,胡麻人,八月瓜,甘草
- 痒自風來，`止癢必先疏風`，故方中以荊芥、防風、牛蒡子、蟬蛻開發腠理透解表風。
- 以蒼朮之辛苦溫，散風祛濕
- `皮膚出現紅疹或如錦紋或如疹塊`，不管有無滲出液，瘙癢甚者均可使用
- 氣血虛弱者慎用，忌食辛辣、魚腥、煙酒、濃茶、咖啡
- 濕疹、風疹、蕁麻疹、德國麻疹、過敏性皮炎、稻田性皮炎、藥物性皮炎、神經性皮炎、水蟲、痱子

### 疏散外風比較
- `偏正頭痛`，或是伴隨鼻塞、流涕，專治「頭風」 => 川芎茶調散
- 口眼歪斜、`手足麻木`，適合體質偏虛的中風先兆或後遺症，專治「肢風」 => 大秦艽湯
- `皮膚瘙癢、紅疹`，專治「肌表之風」 => 消風散


## 平熄內風
- 治療內風引起的疾病，如中風、癲癇、眩暈、肢體麻木、抽搐等症狀，或者內風侵襲經絡引起的疼痛、麻痺、痙攣等症狀。
- 常見於肝風內動、肝陽上亢、肝腎陰虛等證型。

### 羚角鉤藤湯
- 小兒`高熱抽搐`，見高熱，`煩悶躁擾，手足抽搐`，甚至神昏，或妊娠，舌質絳而乾，脈弦而數者
- 涼肝熄風，`增液舒筋`
- 羚羊角,鉤藤,桑葉,川貝母,生地,菊花,茯神,白芍,竹茹,甘草
- 降血壓，鎮靜，抗驚厥，解熱，抗病原微生物，消炎
- 驚厥、流行性乙型腦炎、原發性高血壓、小兒高熱、急驚風、流行性腦脊髓膜炎、妊娠高血壓

### 天麻鉤藤飲
- 肝陽偏亢、`肝風上擾證`、`頭痛眩暈`、易怒失眠、`目赤乾澀`、急躁易怒、口苦、舌紅、苔薄黃、脈弦
- 平肝熄風、`清熱活血`、補益肝腎。
- 天麻,鉤藤,牛膝,石決明,杜仲,黃芩,梔子,益母草,桑寄生,夜交藤,茯神
- 降血壓，鎮靜，調節高級神經活動，抗驚厥，催眠，擴張血管，改善血液循環，抗血栓形成，消炎，解熱。
- 高血壓、腦震盪、腦血栓、面肌痙攣、眩暈、耳鳴、頭痛。

### 大定風珠
- 病後`神倦`，手足蠕動，舌質絳，苔少，脈虛弱者。
- 滋陰熄風。
- 流行性乙型腦炎、化膿性腦膜炎、登革熱、流行性出血熱、敗血症、心力衰竭

### 平熄內風比較
- `高熱抽搐`，適合小兒高熱引起的驚厥，專治「熱風」 => 羚角鉤藤湯
- `頭痛眩暈`，適合肝陽上亢引起的肝風內動，專治「肝風」 => 天麻鉤藤飲
- `神倦`，適合病後體虛的肝風內動，專治「虛風」 => 大定風珠
