# 開竅劑
- 專治因痰濁、熱邪或寒邪阻滯心竅所致的「神昏竅閉」急症
- 芳香走竄，開竅通閉，蘇醒神志
- 適用於神昏、`中風`、高熱驚厥、`癲癇`、`突然昏倒`、`牙關緊閉`等症狀。
- 臨床分為涼開（治熱閉，如安宮牛黃丸）與溫開（治寒閉，如蘇合香丸）兩大類
- 屬`急救治標`之品，不可長期服用，多配伍化痰、清熱或散寒藥物。
  - 僅用於閉證（實證），脫證（虛證、大汗淋漓、肢冷脈微）禁用
  - 孕婦慎用或禁用

- 石膏、寒水石、滑石、磁石、芒硝、硝石、朱砂、玄蔘、升麻、犀角、羚羊角、沉香、木香、甘草、朴硝、硝石、丁香、硃砂、麝香、金箔

- `化濁開竅`，清熱解毒。
