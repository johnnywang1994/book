# VSCode 好用快捷鍵

  1. cmd + , => 打開設置

  2. cmd + p => 打開文件

  3. cmd + l => 選中當前行

  4. ctrl + g => 跳至某行

  5. shift + alt + i => 在所選範圍最後面加入游標（同時編輯）

  6. shift + alt + f => 格式化代碼

  7. cmd + k, + v => 開啟右側 .md 預覽

  8. cmd + k, + z => 全螢幕編輯模式

  9. shift + cmd + l => 選中所有相同內容（加強版 cmd + d）
