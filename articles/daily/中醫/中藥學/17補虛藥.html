# 補虛藥

凡具有补虚扶弱作用，功能治疗人体虚损不足的药物，称为补虚药。又可叫做补益药

  • 主要用于虚症。所谓虚症，一般说来，有气虚、阳虚、血虚、阴虚等不同类型。补虚药根据它的效髓及应用范围，一般也分为补气药、助阳药、养血药、滋阴药
  • 气虚补气，阳虚助阳，血虚养血，阴虚滋阴。但阳虚的，每多包括气虚；而气虚的，常易导致阳虚。气虚和阳虚是表示机体活动能力的衰退。阴虚的每兼血虚；而血虚的，常易导致阴虚。血虚和阴虚是表示体内津液的损耗。这说明人体气血阴阳有着相互依存的关系。因此，益气和助阳，养血和滋阴，又往往相须为用
  • 气血两亏，阴阳俱虚，则补虚药的使用，更须兼筹并顾，灵活掌握，用气血并补或阴阳两补的方法。
  • 补虚药对实邪未尽的病人，应予慎用，以免病邪留滞

# 補氣藥

治疗气虚病症的药物。具有补肺气、益脾气的功效，适用于肺气虚及脾气虚，又常用于血虚的病症，因为气旺可以生血，大失血时，必须运用补气药，因为“有形之血，不能速生；无形之气，所当速固”

  • 脾气虚则神疲倦怠，大便泄泻，食欲不振，脘腹虚胀，甚至浮肿、脱肛
  • 肺气不足，则少气懒言，动作喘乏，易出虚汗

# 救脫補虛實清補-升舉固表健脾化濕-安神瀉火潤肺

  • 人参党参孩儿参-黄耆白朮山藥扁豆-大枣甘草黃精

  • 人参: 甘，平。入脾、肺经

    • 大补元气，补肺益脾，生津，安神，治肺虚气喘、大失血虛脫、脾胃虚弱、倦怠乏力、食欲不振、胸腹胀满，以及久泻脱肛
    • 挽救气虚欲脱之症。临床遇气息短促、汗出肢冷、脉微细，或大量失血引起的虚脱等危急的症候，阳气衰微，又可与附子等同用
    • 治倦怠乏力，气虚脱肛等症，常与黄耆、白朮等配伍
    • 治消渴，热病耗伤津液，生津止渴，故可与生地、天花粉配伍
    • 治神志不安、心悸怔仲、失眠，常与养血安神药如酸枣仁、桂圆肉、当归等同用
    • 高热大汗后，气伤液耗而见身热口渴者，还可与清热泻火药如石膏、知母等同用，这是取它的益气生津作用
    • 人参与袪邪之药同用，可用于邪未清而正气已虚的病症
    • 补气作用较强，一般不用于实症，如外感初起，或里热炽盛，或肝阳上亢，以及湿阻、食滞等引起的胸闷腹胀、便溏泄泻等症，都应忌用

  • 党参: 甘，平。入脾、肺经。

    • 补中益气。治气虚不足，倦怠乏力，气急喘促，脾虚食少，面目浮肿，久泻脱肛
    • 效近人参而较弱，适用于各种气虚不足的病症，在临床上常与黄耆、白朮、山药等配伍
    • 血虚萎黄及慢性出血疾患引起的气血两亏的病症，本品又可配补血药如熟地、当归
    • 可用于虚实相兼之症，如虚火外感，可与解表药同用
    • 本品产量较人参为多，价亦较廉，故在一般补益剂中多用党参；但是如遇虚脱危重之症，急需补气固脱，因党参力薄，当用人参为宜

  • 孩儿参: 甘、微苦，平。入脾、肺经。

    • 补气养胃。病后虚弱，倦怠乏力，饮食减少，心悸，自汗，津少口渴
    • 似人参而力薄，为补气药中一味清补之品
    • 气阴两亏等症，可配合沙参、山药
    • 邪未去尽，而见气虚不足、津少口渴等症，也可应用
    • 本品与清热化痰的药品如鱼腥草、蒲公英、金银花、前胡、瓜萎皮等配伍同用，其疗效较单用清热化痰药为好

  • 黄耆: 甘，微温。入脾、肺经。

    • 补气升阳，固表止汗，托疮生肌，利水退肿。
    • 补气健脾，常与党参、白朮
    • 益气升阳而举陷，常与党参、升麻、柴胡、炙甘草
    • 表虚易感风寒者，可与防风、白朮同用
    • 治水肿而兼有气虚症状者，多配合白朮、茯苓
    • 可与活血袪瘀通络药如当归、川芍、桃仁、红花、地龙等配伍，用于中风偏枯、半身不遂之症

  • 白朮: 苦、甘，温。入脾、胃经

    • 补脾燥湿，利水，止汗，治脾胃虚弱，食少胀满，倦怠乏力，泄泻
    • 补脾胃可与党参、甘草等配伍
    • 健脾燥湿止泻可与陈皮、茯苓
    • 治水湿内停之痰饮或水湿外溢之水肿
      • 治寒饮可与茯苓、桂枝等配伍
      • 治水肿常与茯苓皮、大腹皮
    • 治表虚自汗，与黄耆、浮小麦等同用，有固表止汗之功
    • 有内热者可与黄芩等配伍

  • 山药: 甘，平。入肺、脾经。

    • 补脾胃，益肺肾。治脾胃虚弱，食少体倦，泄泻，肺虚久咳，肾虚梦遗精滑，小便频数，及妇女白带
    • 肺阴不足症状者，可与沙参、麦冬等同用
    • 益肾涩精，如肾亏遗精，则可与熟地、山萸肉、龙骨等配伍

  • 扁豆: 甘，微温。入脾、胃经

    • 健脾化湿。补脾而不滋腻，化湿而不燥烈，因其健脾化湿
    • 用于脾虚泄泻及妇女白带，常与白朮、山药

  • 大枣: 甘，平。入脾经。

    • 补脾胃，养营安神，缓和药性
    • 治脾胃虚弱等症，每与党参、白朮等配伍
    • 养营安神，临床上常与甘草、小麦等同用，以治脏躁症
    • 如与生姜同用，既能协调营卫，又可和理脾胃
    • 脏躁的发病原因，多由情志抑郁或思虑过度，心脾受损，致脏阴不足而成。其临床表现为无故悲伤、精神失常、坐卧不安、心烦不寐(甘麥大棗湯)

  • 甘草: 甘，平。入十二经。

    • 补中益气，泻火解毒，润肺袪痰，缓和药性，缓急定痛
    • 补脾胃不足而益中气，对脾胃虚弱之症，常与党参、白朮、茯苓等补气健脾药配伍
    • 治心血不足、心阳不振之症，可与补血养阴及温通心阳药如阿胶、生地、麦冬、人参、桂枝等品配合
    • 治疮疡肿毒，疮痈肿痛，多与金银花、连翘
    • 对咽喉肿痛，可与桔梗、牛蒡子等配合
    • 治咳嗽喘息等症，常与化痰止咳药配伍应用，不论肺寒咳喘或肺热咳嗽，均可配合
    • 缓和药性，有减低或缓和药物烈性的作用，但湿阻中焦、脘腹胀满者用之能令人气窒满闷，故在使用时必须注意

  • 黄精: 甘，平。入脾、肺经

    • 补脾润肺
    • 治脾胃虚弱，体倦乏力等症，常与党参、白朮等药配合
    • 治肺虚燥咳，常与沙参、天冬、麦冬
    • 可用治糖尿病，常配合山药、黄耆、天花粉、枸杞子

# 助陽藥

助肾阳、益心阳、补脾阳的功能，适用于肾阳不足、心阳不振、脾阳虚弱

  • 肾阳虚则有畏寒、肢冷、阳痿、遗精、遗尿等症
  • 心阳虚则冷汗淋漓、面色恍白、脉细欲绝或出现结代脉
  • 脾阳虚则完谷不化，便搪、泄泻、食欲不振
  • 肾阳衰微不能温运脾阳所引起的泄泻，以及肾气不足，摄纳无权所引起的喘促，都可选用适当的补肾阳药来治疗
  • 心阳虚，可用温里药或补气药治疗。
  • 助阳药性多温燥!!凡有阴虚火旺的症状，应该慎用，以免发生助火劫阴

# 溫補肝腎壯陽-袪风湿,殺蟲-虛喘暖脾-血虛火便秘,平補-虛喘,補肝腎降血壓

  • 鹿茸鹿角韭子-淫羊藿(溫)仙茅(熱)巴戟天蛇床子-补骨脂益智仁-锁阳菟丝子冬虫夏草-胡桃肉續斷杜仲

# 溫不燥補不滯

  • 巴戟天: 補腎陽、袪风湿

  • 菟丝子: 平補腎陰陽、養肝明目

  • 鹿茸（幼）: 甘、咸，温。入肝.肾经。

    • 补督脉，助肾阳，生精髓，强筋骨，适用于肾阳不足、精衰血少及骨软行迟
    • 补益肝肾，调理冲任，固摄带脉，故可止漏束带，用治崩漏带下属于虚寒症状者，可与阿胶、当归、熟地、山萸肉、淮山药、白芍、乌贼骨等配伍

  • 鹿角（老）: 咸，温。入肝、肾经。

    • 温补肝肾、强筋骨、活血消肿。
    • 活血消肿之功，常用于虚寒疮疡之症，良好消散作用，外科常用之品，可配肉桂、白芥子等内服，也可醋磨外用

  • 韭子(韭菜根): 辛、甘，温。入肝、肾经。

    • 温肾壮阳，固精。治阳痿、遗精、遗尿、小便频数，功能止汗。适用于盗汗、虚汗
    • 治肾阳虚衰引起的阳痿、遗精、腰膝痿软、小便频数、遗尿等症，为常用之品，可与枸杞子、五味子、覆盆子、菟丝子

  • 淫羊藿: 辛，温。入肝、肾经。

    • 补肾助阳，袪风湿
    • 治阳痿遗泄，可配仙茅、山萸肉、肉从蓉
    • 治腰膝痿软、肢冷畏寒，可配杜仲，巴戟天、狗脊
    • 治寒湿痹痛、四肢拘挛麻木，可与威灵仙、巴戟天、肉桂、当归、川芎等配伍
    • 仙茅、葫芦巴两药性温偏热，温肾作用较强，服用稍久，即有口苦唇燥，本品则性温而不热，对偏于肾阳虚的患者，久服无不良现象

  • 仙茅: 辛，热。有小毒。入肾经。

    • 温肾壮阳，袪寒除湿。治阳痿精寒，腰膝风冷，筋骨痿痹
    • 辛热性猛，能补命门而兴阳道，除寒湿而暖腰膝，对肾阳不足、命门火衰所致的阳痿精寒、筋骨痿痹等症，常与淫羊茬等配合应用

  • 巴戟天: 甘、辛，微溫，歸腎、肝經

    • 補腎陽，強筋骨，祛風濕
    • 温而不燥，补而不滞，能补肾阳、强筋骨。
    • 治阳痿遗泄，常与肉苁蓉、菟丝子等配合
    • 治腰膝痿软，常与续断、杜仲等配伍
    • 治下肢寒湿痹痛，与附子、狗脊等配合
    • 如属湿热下注、足膝红肿热痛等症，忌用

  • 蛇床子: 辛、苦，温。入肾经。

    • 温肾壮阳，燥湿杀虫。
    • 温肾壮阳之功，对肾虚阳痿及女子不孕等症，常与五味子、菟丝子
    • 外用煎水洗阴部，有杀虫、除湿热的功效。近来用本品制成栓剂，外用治阴道滴虫病
    • 蛇床子汤《医宗金鉴》：蛇床子、威灵仙，当归尾、大黄、苦参，砂壳、葱头。洗阴囊湿痒

  • 补骨脂: 辛、苦，大温。入脾、肾经。

    • 补肾助阳，治虚喘、下元虚冷，阳痿，遗精，早泄，腰部度痛，及小便频数，遗尿
    • 补命门火而温运脾阳，治虚冷泄泻，常与肉豆蔻等同用
    • 温肾而纳气平喘，多与胡桃肉配伍以治虚寒气喘

  • 益智仁: 辛，温。入脾、肾经

    • 补肾固精，缩尿，温脾止泻，摄涎唾
    • 治肾虚遗泄，可与补骨脂、菟丝子等配伍
    • 治尿频、遗尿，可与山药、乌药等配伍
    • 治肾虚白浊或小便余沥，可与川萆薢、乌药等配伍
    • 暖脾止泻的功效，可与党参、白朮、干姜、炙甘草等配伍
    • 脾脏虚寒，不能摄涎，以致口涎多而自流者，本品又能温脾以摄涎，可与党参、茯苓、半夏、陈皮、淮山药等品配伍
    • 脾虚不能摄涎所致，必无口干、口苦的现象；如属脾胃湿热所引起的口涎自流，多有唇赤、口苦、苔黄等症，则宜用黄芩、白芍、甘草等品，不可用辛温的益智仁
    • 益智仁与补骨脂都能温补脾肾，可用于遗精、尿频、遗尿及虚寒泄泻等症。但益智仁偏于温脾固涩，能摄涎唾；补骨脂则偏于补肾助阳，可用于肾虚腰酸及虚喘等症，各有特点

  • 锁阳: 甘，温。入肝、肾经

    • 补肾壮阳益精，润燥滑肠，治老年血虛大便燥结，腰膝软弱
    • 治肾虚阳痿、腰膝无力等症，常与牛膝、枸杞子、山萸肉、五味子、熟地等配
    • 对虚火便秘之症，可与火麻仁、柏子仁等配合

  • 菟丝子: 辛、甘，平。入肝、肾经。

    • 补肾固精，养肝明目
    • 治阳痿遗精，小便频数及肾虚腰痛等症，可与枸杞子、潼蒺藜、杜仲
    • 治肝肾不足、两目昏糊等症，可与枸杞子、女贞子、潼蒺藜
    • 菟丝子性柔润而多液，不温不燥，补而不腻，功能滋补肝肾，为一味平补阴阳的药物，不论属于肾阳虚或肾阴虚，都可应用

  • 冬虫夏草: 甘，温。入肺，肾经

    • 平补阴阳，滋肺补肾，止血化痰
    • 治肺虚劳咳血，常与沙参、麦冬、生地等配合
    • 治阳痿遗精，可与枸杞子、山萸肉、淮山药

  • 胡桃肉: 甘，温。入肺、肾经。

    • 补肾强腰膝，敛肺定喘，润肠通便，便溏腹泻时忌用
    • 治肺肾不足的虚喘，可与人参、补骨脂等配伍

  • 續斷: 苦，微温。入肝、肾经。

    • 补肝肾，强筋骨，续伤折，治崩漏。
    • 治肝肾不足，腰膝瘦痛，脚软乏力，与杜仲配
    • 治筋骨折伤，通利血脉，有接骨疗伤作用，为伤科要药
    • 治崩漏，在临床上常与杜仲、阿胶、当归、地黄、艾叶炭等药配伍

  • 杜仲: 甘，温。入肝、肾经。

    • 补肝肾，强筋骨，安胎。临床上主要用于肾虚腰部酸痛，降血压作用
    • 治肝肾不足，腰膝瘦痛，乏力，眩晕，阳痿，小便频数

# 養血藥

又叫补血药，就是用于治疗血虚病症的药物，面色萎黄、嘴唇及指甲苍白，没有红润的颜色，并且有头晕、耳鸣、心悸、健忘、失眠、月经不调等症

  • 遇血虚兼气虚的，需配用补气药；血虚兼阴虚的，需配用滋阴药
  • 养血药性多粘腻，凡湿浊中阻，腕腹胀满，食少便搪的不宜应用；脾胃虚弱的，应与健胃消化的药物同用，以免影响食欲

# 滋陰補血-活血平肝-補血止血,滋陰,益脾安神

  • 熟地黃桑椹子-當歸白勺-何首乌阿膠桂圆肉

  • 熟地黃：甘，微温。人心、肝、肾经

    • 补血，滋阴
    • 补血滋阴而养肝益肾，凡血虚阴亏、肝肾不足所致的眩晕，均可应用
    • 补血常与当归、白芍等同用
    • 补肝肾常与山茱萸、丹皮等同用
    • 用于心悸、失眠，配党参、酸枣仁、茯苓等品
    • 阴虚火旺、骨蒸潮热等症，可与龟板、知母、黄柏等同用
    • 如血虚、肝肾不足等症而有脾胃运化不良，应用熟地时可与理气健胃的药品如陈皮、砂仁等配伍同用，能减少它滋腻碍胃之性

  • 桑椹子: 甘，寒。人心、肝、肾经。

    • 滋阴补血。治阴血不足，眩晕，失眠，以及肝肾阴虚，须发早白
    • 益肝肾、养阴血，对于阴血不足的眩晕、失眠等症，常与熟地黄、白芍等配合
    • 对肝肾不足、须发早白、耳聋目昏，又可配合何首乌、女贞子

  • 當歸：甘、辛，温。入肝、心、脾经。

    • 补血调经，活血止痛。
    • 当归功能补血，常与黄耆、党参等配伍，用治血虚体弱；因它又能活血，故可用于调经，为妇科常用时药品
    • 治经行腹痛，常与香附、延胡索等同用；治经闭不通，可与桃仁、红花等配伍；治崩漏，可与阿胶、地黄、艾叶等同用
    • 活血治跌打损伤瘀痛，痈肿血滞疼痛，产后瘀滞腹痛，风湿痹痛及经络不利
    • 治损伤瘀痛，可与红花、桃仁、落得打等品配伍
    • 治痈肿瘀滞疼痛，在肿疡期，可与银花、连翘、丹皮、赤芍、甘草等配伍

  • 白芍: 苦、酸，微寒。入肝经。

    • 养血敛阴，柔肝止痛，平肝阳。
    • 治月经不调，经行腹痛，崩漏，以及自汗、盗汗
    • 与桂枝同用，能协调营卫，用以治疗外感风寒、表虚自汗而恶风
    • 养血而柔肝，缓急而止痛，故可用于肝气不和所致的胸胁疼痛、腹痛及手足拘挛等症

  • 阿胶: 甘，平。入肺、肝、肾经

    • 补血止血，滋阴润肺。
    • 补血作用较佳，为治血虚的要药，常配伍当归、党参、黄耆等同用
    • 善于止血，对一切失血之症，均可应用，然以咯血、便血、崩漏等用之较为适宜
    • 治热病伤阴，虚烦不眠，滋阴而润燥，对热病伤阴，内风欲动，常配合钩藤、牡蛎等同用；对阴亏火炽、虚烦不眠，常配合白芍、黄连

  • 何首乌: 苦、涩，微温。制熟则味兼甘。入肝、肾经

    • 补肝肾，益精血，润肠通便，解毒，截疟
    • 补肝肾作用显着，又有补血作用，用于血虚萎黄、头晕目眩，头发早白、腰膝酸软等症，常与地黄、枸杞子、菟丝子
    • 润肠通便、消疮毒，配连翘、玄参等能解毒消痈
    • 现代临床上应用制首乌，已有所发展，如用治高血压、血管硬化、头晕等症，常与桑寄生、女贞子等配伍；用治冠状动脉硬化性心脏病，常与丹参、郁金、瓜萎等配伍

  • 桂圆肉: 甘，温。入心、脾经。

    • 补心安神，养血益脾，治心脾虚损的失眠健忘，惊悸怔仲
    • 治气血不足，体虚力弱，补脾胃之气，又能补营血不足

# 滋陰藥

能治疗阴虚病症的药物，滋肾阴、补肺阴、养胃阴、益肝阴等功效，适用于肾阴不足、肺阴虚弱、胃阴耗损、肝阴亏乏，滋阴药各有专长，应随症选用

  • 肺阴虚干咳，咯血，虚热，烦渴
  • 胃阴虚　唇赤，舌绛，苔剥，津少口渴，或不知饥饿，或胃中虚嘈，甚或有呕、秽
  • 肝阴虚　两眼干涩昏花，眩晕
  • 肾阴虚　潮热，盗汗或遗精

大多甘寒滋腻，如遇脾肾阳虚，痰湿内阻，胸闷食少，便溏腹胀等症，不宜应用

# 潤肺清心滋腎-養胃潤肺安神-補肝腎-凉血止血清热解毒

  • 沙参麥門冬天門冬-石斛玉竹百合-楮实子枸杞子女贞子-旱莲草羊乳根

  • 沙参(珠儿参、西洋参): 甘，微寒。入肺、胃经

    • 润肺止咳，养胃生津，甘凉柔润，治肺虚有热、干咳少痰，或久咳声哑
    • 治肺虚热咳的要药，常与川贝、麦冬等配伍
    • 养胃阴而复津液，故可用于热病伤津、舌绛口渴等症，常与麦冬、生地、石斛
    • 南沙参偏于清肺袪痰，养胃生津的作用较差；北沙参养胃生津的作用较佳

  • 麥門冬(清心): 甘、微苦，微寒。入心、肺、胃经

    • 清心润肺，养胃生津
    • 治肺虚热咳，咯血等症，可与沙参、天冬、生地等配伍
    • 治津少口渴
    • 清热润燥滑肠之功，与玄参相似，两药常相须配合，用于热病伤津、肠燥便秘。如属脾胃虚寒，大便溏泻或有湿滞者，不宜应用
    • 如胃阴不足、心烦燥渴等症，多用麦冬；肾阴亏损、潮热遗精等症，则多用天冬

  • 天门冬(滋腎): 甘、苦，大寒。入肺、肾经

    • 润肺止咳，养阴生津
    • 治肺阴受伤，燥咳、咯血，常与麦冬、沙参、生地
    • 治阴虚内热，口渴，凡熱病傷陰，与生地、麦冬、石斛
    • 善治肺肾虚热。用于上焦，能清肺热而养肺阴；用于下焦，能滋肾养阴，且可润燥滑肠。
    • 如属脾胃虚弱泄泻者，不宜应用
    • 大寒治腎陰虛潮熱

  • 石斛（ㄏㄨˊ）(養胃): 甘，微寒。入肺、胃、肾经

    • 滋阴，养胃，生津，用于热病伤阴，口干燥渴，或病后津亏虚热
    • 常与麦冬、沙参、生地等品配伍
    • 养胃生津之功较麦冬为佳，但无润肺止咳、清心除烦的作用

  • 玉竹(润肺養胃): 甘，平。入肺、胃经

    • 滋阴润肺，养胃生津
    • 润肺养胃、生津增液的功效，适用于肺胃燥热之症，常与沙参、麦冬、天冬等配伍

  • 百合: 甘，微寒。入心、肺经

    • 润肺止咳，宁心安神
    • 治肺燥或肺热咳嗽等症，常与麦冬、沙参、贝母、甘草
    • 寧心安神作用，用于热病后余热未清、神思恍惚之症，与知母、地黄等配合

  • 楮实子: 甘，寒。入脾、肾经

    • 补肾强筋骨，明目，利尿
    • 治肾虚阳痿、腰酸，常与熟地、枸杞、苁蓉、怀牛膝
    • 养肝明目，可用于血虚头晕、眼花
    • 治水肿利尿，可配合冬瓜皮、赤小豆

  • 枸杞子: 甘，平。入肝、肾经。

    • 补肾益精，养肝明目
    • 不论肾阴虚亏或肾阳不足，皆可应用
    • 补养肝肾的药品，功与潼蒺藜、菟丝子相近似，但後两药助阳之功胜于养阴，故归入助阳药，枸杞子则滋阴之功胜于助阳
    • 配以熟地，则补肝滋肾；配以菊花，则养肝明目；配以黄精，则补精益气
    • 外邪实热、脾虚湿滞及肠滑便溏者，不宜应用

  • 女贞子: 甘、苦，平。入肝、肾经

    • 补肾滋阴，养肝明目，治肝肾不足，头晕，耳鸣，两目昏糊，头发早白
    • 滋养肝肾的功用与枸杞子、桑椹子相近，但女贞子，只能滋阴，不能助阳，偏治肝肾阴虚，并无补血作用

  • 旱莲草: 甘、酸，寒。入肝、肾经

    • 养阴益肾，凉血止血的作用颇佳，内服、外用，都有功效

  • 羊乳根: 甘，平。
